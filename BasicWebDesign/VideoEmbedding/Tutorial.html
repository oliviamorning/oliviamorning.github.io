Video Embedding Tutorial - Olivia Morning

Step 1 - Beginning

The first thing you want to do when embedding a video into your website is to have the proper code to begin. At the top, I have basic HTML code to get started. This incluede the DOCTYPE, the language, the header with a title embeded in it, and the body element.

Step 2 - Getting Your Video

The second step is to find the video you want to use.

Option 1

Many popular video services have an option to automatically copy code they created for you.

To do this, click on the video's "Share" button, and then "Embeded" or >/<. In my example, I am using Vimeo.

After that, you should see a window like this.

Option 2

The other way to obtain a video is by doing the code manually. This is helpful if the website you are using does not generate the code for you.

Using the same video as an example, all you have to do is copy the video link by going to "Share", and then copying the link.

Step 3 - Coding

The most inportant step-the actual code!

Option 1

If you went for Option 1 in Step 2, you're almost done! Once you copy and paste the code into your text editor, all you have to do is finish the code from the beginning, like your body and HTML tags. After that, you're all set!

Option 2

If the video you chose didn't generate code for you, there's no need to worry! It's simple to code.

The first thing you want to code is the <iframe></iframe> tag. Inside this tag, you should have the source (src), height, width and title attributes.

The "source" (or src) attribute is where you want to paste the link you got from the video in Step 2.

For the "height" and "width" attributes, you can make it whatever size you want.

***Note: the bigger you make the video, the longer loading time can be for your website viewers.

The last thing you should have is a title. This just simply lets your viewers know what the video is.

Conclusion

Now you should have a fully embedded video, just like this one!
